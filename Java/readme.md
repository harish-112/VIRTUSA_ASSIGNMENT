- **Core Java Concepts** - Variables, Data Types, Control Flow, Loops, Conditionals
- **Collection Framework** - HashMap, ArrayList for data management
- **Exception Handling** - Input validation, error handling, null checks
- **User Interface** - Scanner-based console interaction


## Project Overview

This is a console-based Banking System that demonstrates core Java concepts through a practical application. Users can create accounts, perform transactions, and view their account details through an interactive menu-driven interface.

## Features

1. **Create Account** - Create new bank accounts with account number, holder name, account type, initial balance, and password
2. **Deposit Money** - Add funds to an account after authentication
3. **Withdraw Money** - Withdraw funds with validation for sufficient balance
4. **Transfer Money** - Transfer funds between two accounts
5. **Check Balance** - View current account balance after login
6. **Transaction History** - View complete transaction records for an account
7. **Password Protected** - Each account requires authentication before performing operations

## Classes Used

- **BankingSystem.java** - Main class with menu-driven interface and user interaction
- **Bank.java** - Manages all banking operations, handles account storage, and business logic
- **Account.java** - Represents individual bank accounts with account details and transaction tracking

## Key Concepts Applied

- **OOP Principles** - Encapsulation (private data members), Classes and Objects
- **Collections** - HashMap for account storage, ArrayList for transaction history
- **Core Java** - Scanner for input, string handling, loops, conditionals
- **Exception Handling** - Input validation, null checks, password verification
- **Authentication** - Password-based login system for account security

## How to Run

1. Compile all Java files
2. Run BankingSystem.java
3. Select options from the menu to perform operations
4. Provide required inputs when prompted
